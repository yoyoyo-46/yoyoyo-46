C言語でオセロを作る

ー Part.1 ー

到達目標の確認から、盤面を表示させる関数 "bord関数" の完成まで。

はじめに

今回作るオセロは、隣の人が対戦相手、駒が置けるところにポインターを表示（コレ →・）駒を置くときは座標指定、です。ほかにも制限はありますが、その時々で説明します。

下準備

では早速ですが作っていきましょう。オセロを作るにあたって絶対欠かせないのが二次元配列です。今回も勿論これを使用します。配列名をblock 要素数を8とした二次元配列をint型で宣言します。


				1	#include<stdio.h>
				2	int main(void){
				3		int block[8][8];
				4
				5		return 0;
				6	}

これで下準備は完了です。

盤面の表示

ここからは本格的に作業に入ります。まずオセロの盤面がないことには何も始まらないので、盤面を表示する関数を作っていきましょう。関数名をbord 仮引数をblock配列にし、void型で宣言します。今回は使う関数がとても多いので全てプロトタイプ宣言とします。


				1	#include<stdio.h>
				2
				3	void bord(int block[][8]);
				4
				5	int main(void){
				6		int block[8][8];
				7
				8		return 0;
				9	}
